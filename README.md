## 🚀 Overview

**Readify** is a professional, high-performance web application designed to help developers create stunning, high-traffic GitHub README profiles in minutes. Gone are the days of manual markdown tweaking—Readify provides a beautiful, interactive editor with live preview, 200+ skill badges, and deep integration with GitHub Stats and WakaTime.

### 🌟 Key Features

- **🎯 Tailored Templates**: Choose between **Student**, **Professional**, or **Expert** profiles, each with field-specific sections (Grants, Research, OSS, Conference Speaking, etc.).
- **⚡ Live Preview**: High-fidelity markdown rendering that updates instantly as you type.
- **🎨 200+ Tech Skill Badges**: Searchable database of professionally styled tech badges.
- **⏱️ WakaTime Coding Stats**: (Star Feature) Embed real-time coding activity graphs from your IDE.
- **🔥 GitHub Stats & Streaks**: Beautifully themed cards for your commits, PRs, issues, and contribution streaks.
- **🌊 Animated Components**: Capsule-render wave headers, typing animations, and professional dividers.
- **📱 Fully Responsive**: Generate your README on any device with our mobile-optimized editor.
- **🌗 10+ Premium Themes**: Toggle between Tokyo Night, Radical, Dracula, Midnight Purple, and more.

## 🚦 Getting Started

### 1. Installation

Clone the repository and install dependencies:

```bash
git clone https://github.com/MathuR-A01/Readify-profile-generator.git
cd gitglow
npm install
```

### 2. Development

Run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) to see the app.

## 📈 WakaTime Integration Guide

To enable real-time coding stats:
1. Sign up for a free account at [WakaTime](https://wakatime.com).
2. Install the WakaTime plugin in your IDE (VS Code, JetBrains, etc.).
3. Copy your API key from your [Account Settings](https://wakatime.com/settings/account).
4. Enter your WakaTime username in the **WakaTime Step** (Step 7) in Readify.
5. Your README will now show live daily coding time and language breakdown!
